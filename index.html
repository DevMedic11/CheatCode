Getting to know CSS Selectors!

Selectors are used to style HTML elements according to their type and/or attributes.

Type Syntax Description Example
Element element-name { } Use the name/tag (ie. h1) to apply styles to all associated content. p { color: red; }
Class .class-name { } Using "class ='name'" in your HTML tag, will apply CSS to all associated content. .specialtext { font-weight: bold; }
ID #id-name Using "id='name'" in your HTML tag, will apply specified CSS to associated content #specialpic { border-color: pink;}
Attributes element[attribute='value'] After placing an attribute on an HTML tag, you can use the attribute and value to apply specified CSS to the content. div[type='clear'] { opacity: 0.6;}

But wait...Theres more!

You can also assign CSS to elements only with a specific class value

Syntax Example
element.class h3.black { color: black;}
